# KiglerScrape

A personal web platform for curating web links, styled after Reddit.

When you add a URL, the app automatically:
- **Fetches the page** to extract the title and thumbnail
- **Generates an AI summary** (2-3 sentences) using Claude
- **Auto-tags the content** with 3-6 relevant tags

---

## Features

| Feature | Details |
|---|---|
| Reddit-style UI | Upvote/downvote, sort by New / Top / Hot, tag chips |
| AI-powered metadata | Title, summary & tags generated by Claude |
| Search | Real-time client-side search across titles, summaries & tags |
| Tag filtering | Click any tag in the sidebar or on a card to filter |
| Persistent storage | SQLite database — no external services needed |

---

## Quick Start

### 1. Install Node.js
Download from https://nodejs.org (LTS version recommended)

### 2. Clone the repo
```bash
git clone https://github.com/mkigler-papa/KiglerScrape.git
cd KiglerScrape
```

### 3. Install dependencies
```bash
npm install
```

### 4. Set up your API key
```bash
# Windows
copy .env.example .env

# Mac / Linux
cp .env.example .env
```

Open `.env` in any text editor and replace `your_anthropic_api_key_here` with your key from https://console.anthropic.com/

### 5. Start the server
```bash
npm start
```

Open **http://localhost:3000** in your browser — done!

## How to use

1. Click **"+ Add Link"** (top right or sidebar)
2. Paste any URL and press Enter or click **Add Link**
3. Wait ~5-10 seconds while the AI processes the page
4. Your link appears with a title, summary, and tags

Click any **tag chip** to filter links by topic.  
Use the **sort buttons** (New / Top / Hot) to reorder your feed.  
Use the **search bar** to find links by keyword.

---

## Tech stack
- **Backend**: Node.js + Express
- **Database**: SQLite (via `better-sqlite3`)
- **Scraping**: Axios + Cheerio
- **AI**: Anthropic Claude
- **Frontend**: Vanilla HTML / CSS / JavaScript
